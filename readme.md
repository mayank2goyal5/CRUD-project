************Python CRUD Project************

=============================================================
Overview

This project demonstrates a basic CRUD (Create, Read, Update, Delete) application built using Python. 
The program interacts with a database and performs fundamental data operations through Python scripts and SQL queries.

CRUD operations are the foundation of most data‑driven applications such as web apps, desktop software, and enterprise systems.

================================================================
Features
**Create new records in the database
**Read / retrieve stored records
**Update existing records
**Delete records from the database
**Simple Python implementation for learning database interaction
**Technologies Used
**Python
**SQL
**Database connectivity (via Python database connector)
**Project Structure

==========================================================================
CRUD-project/
│
├── main.py           # Main program file
├── database.py       # Database connection logic
├── crud_functions.py # Functions for create, read, update, delete
└── other files...

******************How It Works**********************
---->The program connects to the database.
---->Users can perform CRUD operations using Python functions.
---->SQL queries are executed to manipulate database records.
---->The database returns results which are displayed in the console.

Example Operations
---->Insert new data into the database
---->Fetch all records
---->Update an existing record
---->Delete a record from the table
---->Learning Purpose

============================================================================
This project was created to show functionality of

**Python database connectivity
**SQL query execution in Python
**Implementation of CRUD operations
